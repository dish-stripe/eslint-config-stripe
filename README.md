# eslint-config-stripe

This package provides Stripe's .eslintrc as an extensible shared config.

### Installation

```
$ npm install --save-dev eslint-config-stripe
```

### Usage

Add `extends: 'stripe'` to your `.eslintrc`:

(.eslintrc.yml)
```yaml
---
extends: 'stripe'
```

(.eslintrc)
```json
{
"extends": "stripe"
}
```
